# Binary Tree Question
### 지원


### 1. 트리 자료구조에 대해 설명하시오
트리는 계층적 데이터를 나타내는 노드들의 집합이다.
노드와 간선으로 이뤄진 비선형 자료구조로 무방향이며 사이클이 없는 그래프이다.
가장 중요한 속성은 루트 노드를 제외한 모든 노드는 단 하나의 부모 노드를 갖는다.
   
### 2. 트리의 순회 방식 종류
1. 전위 순회 : 루트 -> 왼쪽 자식 -> 오른쪽 자식
2. 중위 순회 : 왼쪽 자식 -> 루트 -> 오른쪽 자식
3. 후위 순회 : 왼쪽 자식 -> 오른쪽 자식 -> 루트

### 3. 이진트리는 어떤 자료구조인가?
모든 노드의 자식 개수가 2개 이하인 트리

### 4. BST는 어떤 자료구조인가?
어떤 노드를 선택하든 해당 노드의 왼쪽에는 작은 값의 노드로 이루어지고 
오른쪽은 큰 값들로만 이루어져있는 이진트리이다.
트리가 편향된 경우 발생할 수 있는 O(N)을 개선하기 위한 자료구조다.
트리에서 삽입, 삭제가 일어날 때 왼쪽, 오른쪽 자식의 높이 차가 1 이하가 되도록 균형을 맞춘다.
따라서 검색, 저장, 삭제의 시간 복잡도는 O(logN)을 유지한다.
대표적으로 AVL 트리와 Red-Black 트리가 있다.

### 5. BST의 시간 복잡도는 어떻게 되는가?
검색, 저장, 삭제 모두 O(logN)이고 최악은 O(n)이다.
최악은 균형이 깨져 한 쪽으로 치우쳐진 경우인데 이렇게되면 Linked List와 동일하여 O(n)이 된다.

### 6. BST의 최악의 경우 해결 방법은?
자가 균형 이진 탐색트리(BST)는 알고리즘을 사용해 이진 트리의 균형이 잘 맞도록 유지하여
높이를 가능한 낮게 유지할 수 있다. 대표적으로 AVL 트리, Red-Black 트리가 있다.

### 7. Red-Black 트리란?
각 노드에 빨간색, 검은색을 갖게 하고 삽입, 삭제 시 규칙을 갖도록하여 균형을 유지한다.
루트, 말단 노드는 모두 Black (Root, Leaf)
Red의 자식은 모두 Black이어야한다.
AVL 보다 느슨한 균형으로 삽입, 삭제가 빠르다

### 8. AVL 트리란?
트리에 노드 삽입, 삭제 시 자가 균형 조정 메서드를 호출하여 균형을 유지한다.
왼쪽, 오른쪽 높이 차이(BF, Balance Factor)가 1보다 커지면 회전을 통해 유지한다.
